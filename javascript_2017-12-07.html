So, a pretty common function (callback) to be in your reduce() method is summing up the numbers in the array you're reducing.Seems legit.

        const years = [1982, 1994, 2015, 2002, 1945, 1861];
        years.reduce(function (sum, value) {
          // I like console.log statements when I'm messing around.
        	console.log("sum: " + sum + "; value: " + value);
        	let temp = sum+value;
        	console.log("sum so far: " + temp);
        	return sum+value;
        }, 0);

There's also the arrow-function version, but that allows for slightly less console.log().

      const years = [1982, 1994, 2015, 2002, 1945, 1861];
      years.reduce((sum, value) => sum + value, 1);

It's worth noting that in the first version, I didn't provide an InitialValue, so it's assumed to be 0 and returns a value of 117999, whereas the arrow function version starts with 1 and returns 118000.
